# Description
Fully responsive website with modern UI and UX in ReactJS.
## What's the tools have I used?
- TailwindCss
- React
- Postcss
# Install
Clone the repository: 
```
git clone https://github.com/Nikaty/modern-bank.git
```
Install dependences:
```
npm install
```
or
```
npm i
```
# How to run
```
npm start
```
